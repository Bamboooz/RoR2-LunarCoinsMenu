# LunarCoinsMenu
A Risk of Rain 2 mod that allows you to set your lunar coins to a certain value once, or at the start of each run using Risk of Options.

In Risk of Options settings, go to LunarCoinsMenu, you will have a coins amount slider, and a mode selector.
Select Disabled if you don't want your coins to update (default)
Select Once to update the coins once on the start of the next run (will automatically switch to disabled)
Select OnEachRun to update at the start of each run
